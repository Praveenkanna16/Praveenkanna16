# 🌌 GitHub Profile Transformation Guide

This guide explains how to completely transform your GitHub profile from a default layout (Image 1) to the custom, premium dark-mode, lo-fi aesthetic layout (Image 2).

---

## 🛠️ Step-by-Step Instructions

### Step 1: Create a Special Repository
1. Go to GitHub and log in.
2. Create a new repository by clicking the `+` icon in the top right or navigating to: [https://github.com/new](https://github.com/new).
3. **Repository Name:** Enter your exact username: `Praveenkanna16`
4. **Description:** Optional (e.g., "Praveen's special profile repository").
5. **Visibility:** Must be set to **Public**.
6. **Initialize:** Check the box that says **Add a README file** (or leave it unchecked and push the provided `README.md`).
7. Click **Create repository**.

> [!NOTE]  
> GitHub will show a special green box indicating that this is a special repository that will display on your public profile!
